# FINANCETRAKER2
Finance Tracker is a desktop application built with C# and Windows Forms that helps users manage their personal finances. 
Users can record income and expenses, categorize transactions, and view financial records in a DataGridView.
## Features

- Add income and expense records
- Categorize transactions
- Display records in a DataGridView
- Save and load financial data
- Simple and user-friendly interface
- ## Technologies Used

- C#
- Windows Forms
- .NET
- File handling (text files)
- ## How to Run

1. Clone the repository
2. Open the project in Visual Studio
3. Build and run the application
